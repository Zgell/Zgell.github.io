To-Do List

This page will list some of the ideas for projects that I would like to try in the future. Some of them are really ambitious and may never get finished, but the list serves as a guideline for some of the projects I should at least attempt.

The list below is in no particular order, I have only numbered it so that readers can get a sense for how many project ideas I have.

  1. Upgraded Retro Gaming Console
    • A twist on an old project I made when I was 13 years old. I created a mini retro gaming console using a Raspberry Pi and a program called "RetroPie", which allowed the Raspberry Pi to emulate a bunch of old ROMs. Ever since I made that console, I've wanted to make a more powerful one that can run emulators for newer consoles since the Raspberry Pi (at the time) did not have a lot of raw compute power.
    • One approach I've thought of would be to take an ARM-based chromebook, gut it and use the base system as it has 8 CPU cores and most likely a more powerful GPU. I could also use the screen for something else (like a DIY monitor to go with it).
  2. Advanced version of Conway's "Game of Life" Simulation
    • Literally just a simulation of the popular cellular automaton "Game of Life" but with some extra features
  3. Microwave Gun
    • May or may not do this one due to a number of problems (health and safety, cost, ethics) but it's still one that has piqued my interests.
  4. Coilgun, Railgun or a combination of the two
    • The hybrid design I have in mind is a coilgun design to get the projectile moving initially after being chambered, and then the railgun barrel component would accelerate the projectile.
  5. Bioreactor
    • I'd like to grow some bioluminescent bacteria as an experiment, if not for decoration.
  6. A Custom Hydroponic Garden
    • I'd love to use something like an Arduino to automate a garden for a summer project. It might also be fun to make it solar-powered and completely autonomous.
  7. Arc Lighter
    • I think having an electric igniter is pretty cool and I'd like to try my hand at making one to see if it's any more useful than a regular lighter. I've seen people making them online but I've never seen a comparison between the two kinds.
  8. 3D Chess Simulation
    • I know that 3D chess is a thing that has been created, but I'm unaware of any software simulations of this game. I would like to create one to get a grip on both graphics and also creating a smart "AI" to play against.
  9. 3D LED Cube
    • It would be a neat little decoration for my desk or something. It could also be neat to have a 3D version of a cellular automaton going on it.
  10. Something to do with CRISPR technology
    • This could relate to the bioreactor project, but I think that doing a small experiment with CRISPR would be really neat.
    • This is one of those things that's really hard right now but in the future may be more accessible, so it's not exactly at the top of my priorities list.
  11. A custom 4-Bit/8-Bit CPU
    • It would be a lot of fun to design a custom computer system from the ground up, although it would be a complicated task.
  12. A Crude Radio Circuit
    • I think that building a small crude radio would be a great experiment to learn more about circuits (which at the moment I am not very experienced with).
  13. A High-Power Laser
    • Although I'd want it mostly for fun, there's a lot that you can do with a powerful laser.
    • It's possible that I may use this as an anti-bird defense system for my house as at my current home we have some issues with the magpies that live around here.
  14. ESP8266-Powered Digital Weather and Environment Display System
    • I really like the ESP8266 processor and I'd like to make much more with it than just a little Wi-Fi device. I think that a weather display would be an easy and practical device that would also give me experience with screens and such.
  15. Directional Speaker / "Sound Laser"
    • I could see a lot of use for a device that allows you to broadcast sound to a specific area, and I have several ideas for projects that come from it. But for now, the speaker itself would be interesting enough to do.
  16. A Robot, Powered by an Arduino or a Raspberry Pi
    • I'd like more experience with robotics and some basic robot with either of these devices would be great for that. I've also wanted to have a general-purpose robot for other projects and automation stuff.
  17. A Dedicated Server Computer
    • There's several reasons I'd want a proper server computer, but some of the main ones are for running private servers for friends in video games or for handling cloud-based applications.
  18. Discord Bot Expansion
    • I've made a few basic Discord bots in the past using the Discord.py library, but I'd also like to make a fully-featured one that could be used in a server.
  19. Graphene Synthesis
    • Graphene is a material that I find really interesting, and I'd love to be able to make even a crude sample of it just to play around with it. Synthesizing some very crude stuff doesn't seem terribly difficult.
  20. A 4D Game/Graphical Engine
    • My main inspiration for this idea was a game called "Miegakure", which was a game that took place in 4 spatial dimensions that was never released. I think that making my own attempt at such a system would be very interesting.
  21. A Voxel Engine for a Minecraft Clone
    • I'd be interested in making a Minecraft clone in order to understand how to optimize voxel-based engines, world generation, and noise algorithms.
  22. An Emulator for the NES or for the Commodore 64
    • Writing an emulator in C++ would be a really great experiment for me to understand basic computer systems as well as strengthen my knowledge in complex C++ programs.
    • One of my choices for this project would be the NES as it's one of the most fun systems to emulate in my opinion as it has a good selection of classic video games.
    • The other system I'd like to try to emulate would be the Commodore 64, which I have a personal connection with since I inherited one from my late grandparents.
  23. An AI-Based Card Game.
    • How it would work is that we would use those old custom Pokemon cards that were popular around 2010 as the source. The program would pull a bunch of random images from the internet and use an AI program to turn those into playable cards (it would generate stats and attacks for each card) that would be used in a card game.
  24. Turning a Mobile VR Headset into a Proper VR Headset
    • I got a mobile VR headset from my parents one christmas, and as neat as it is, it's not terribly useful and is definitely dwarfed by all of the PC headsets. However, I think that you could turn one of these headsets into a decent PC headset for casual use as long as you wrote the write software for it and added the right hardware.
  25. A Music Streaming Service and/or App
    • I came up with this idea from all of the frustrations I had suffered from platforms such as Spotify and Soundcloud. I thought it would be a neat experiment that would build my experience in web stuff if I were to come up with my own solution for music.
  26. My Own Cryptographic Algorithm
    • I want to make a really unique cryptographic algorithm for obscuring information, and since the format will be very different and obscure, most people will not know how to crack it.
  27. A Minecraft Mod
    • A mod for Minecraft would be great as it is a video game that I really enjoyed as a kid, but it's also a really good task to practice my skills in the Java programming language.
