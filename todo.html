To-Do List

This page will list some of the ideas for projects that I would like to try in the future. Some of them are really ambitious and may never get finished, but the list serves as a guideline for some of the projects I should at least attempt.

The list below is in no particular order, I have only numbered it so that readers can get a sense for how many project ideas I have.

Please take the following list with a massive grain of salt.

Software Projects

  • An updated project portfolio
  • Learn a graphics library, such as OpenGL or Vulkan
  • Write a program/game for an old game console
    • Candidates include the NES, Gameboy or Gameboy Advance
  • A machine learning project
  • Write my own programming language
  • Write some more Python-based web scrapers and web automation scripts
  • A Google Maps clone, run entirely on the client's computer, powered by OSM data.
  • Instagram Profile Picture Switcher Script
    • Side note: a script for this actually does exist on GitHub already, but it's outdated as the Instagram API has been updated since it was released. Since there is no longer a working script on GitHub, I would like to make one for anyone interested in it.
  • Writing my own game hack (for educational purposes)
  • A proper, fully detailed Discord bot
    • I've written a couple of small ones in the past, but nothing serious thus far.
  • A 4-dimensional game engine, inspired by Miegakure
  • An emulator for some system
    • Some reasonable picks include the NES, Commodore 64 or the Gameboy.
    • I've also strongly considered writing one for the ARM Cortex M4 microcontroller since a good one doesn't exist yet.
  • A custom music streaming application
    • Mostly born out of frustration for platforms like Soundcloud and Spotify, this application would not be its own platform, but rather allow you to stream music from multiple platforms as well as add some additional features.
  • A mod for Minecraft
  • My own fork of OpenRocket, a rocket simulator.
