Pokemon Teambuilder Algorithm

An algorithm I created to help you build the best Pokemon team.

  • Started On: January 5th, 2020
  • Finished On: January 17th, 2020 (development on pause)
  • Documented On: January 17th, 2020
  1. Introduction

Introduction

Back when I was younger, I used to participate in competitive Pokemon battles. One important part of these competitive battles was the process of assembling the perfect team. Typically people do this by hand by picking a specific Pokemon that they want to use, and then choosing Pokemon that best compliment the previous one in regards to type match-ups and different roles that each one plays. This process isn't unreasonably difficult, but one thing I've always been curious about is whether or not you could automate this. It always seemed like it would be really difficult because there were so many factors to consider in order to pick the optimal solution, but I've always thought that it was possible.

Fast forward to the present, and this is my attempt at such an algorithm. This page will include all of the iterations of this idea, my thought processes behind them, and the results of each one. Each iteration of the algorithm uses different parameters to evaluate the best type combination. I will update this page as I make new versions of this algorithm, as it will take a while to make a final version that I am satisfied with.
