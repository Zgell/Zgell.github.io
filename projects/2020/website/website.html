Explanation

As I have listed in other sections of the website, this site is intended to serve two main purposes:

  1. I want to store and document all of my projects in a convenient place
  2. I want to display all of my work for those who are interested

I created the website by first looking into how to host the site. I have been looking around the internet for the best way to host a website for free that gives me the most control over the content of the site. I had originally created a private site on weebly.com to hold information, but I wasn't happy with the formatting and so I gave up on it. I then tried to create a private site using Google Sites but the new version of the service lacks a lot of basic web design features.

After all of that, I finally discovered Github Pages, which allows you to host your own website through Github. I figured that Github Pages would be perfect since I could also use Github to host some of my code on it, so it could serve two purposes. It also gave me full control over the site since you have to upload HTML to host it, but there was a problem: I was far from experienced with HTML, CSS and JavaScript.

After learning about Github Pages through some YouTube tutorials, I found some online websites that offer free to use HTML templates for websites that I could use as a reference to figure out how to make a website. I downloaded a few templates but eventually chose one that I liked the most, and I built this website off of that one.

A lot of my very minimal knowledge of CSS came from this project. Since I haven't done much web development, I haven't ever found myself with the need for experience in more web-focused languages such as CSS or JavaScript.
